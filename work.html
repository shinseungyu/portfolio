  • insgight 1

    접근성 가이드라인을 준수한 프로젝트를 직접 수행하여,
    모든 사용자가 차별 없이 이용할 수 있는 환경을 구축

    모든 사용자가 차별 없이 서비스를 이용할 수 있도록 WCAG 및 한국형 접근성 가이드라인을 반영하여 포용적 환경을 구현했습니다.

    국제 웹 콘텐츠 접근성 지침(WCAG)과 한국형 웹 접근성 지침을 종합적으로 참고하여 설계 및 구현에 반영하였습니다.

    다양한 기기 환경을 고려해 모바일 특화 접근성 요소를 세부적으로 검토하고 적용하였습니다.

  • insight 2


    숨김 텍스트 활용을 통해 접근성을 강화하고,
    입력폼에 aria-label 및 label/for 속성을 적용

    웹 요소별 의미와 역할을 명확히 정의하고, 숨김 텍스트를 활용하여 스크린리더 사용자를 고려한 인터랙션을 구현했습니다. 또한 입력폼에는 aria-label과 label/for 속성을 적용하여 모든 사용자가 편리하게 정보를 입력할 수 있도록 대응했습니다.

    버튼이 아닌 요소에도 button 역할을 명시적으로 지정하여 접근성 지원 도구가 올바르게 인식하도록 구현했습니다.

    숨김 텍스트(hidden text)를 적절히 활용하고, input 폼에는 label/for 및 aria-label 속성을 적용하여 사용자의 입력 편의성과 접근성을 동시에 확보했습니다.

  • insight 3

    포커스 관리를 체계적으로 적용하여
    사용자 경험과 접근성을 개선

    불필요한 포커스를 차단하고, 스와이퍼(Swiper)와 같은 UI 컴포넌트 내에서 포커스 이동을 제어하여 모든 사용자가 효율적이고 직관적으로 인터랙션할 수 있도록 환경을 구현했습니다.

    aria-hidden="true" 속성을 활용하여 화면 내 불필요한 요소에 포커스가 가지 않도록 차단하고, 스크린리더 사용자가 혼동 없이 콘텐츠를 탐색할 수 있도록 했습니다.

    Swiper 등 슬라이드 UI 내에서는 탭 키 이동 및 포커스 순서를 제어하여, 사용자가 의도하지 않은 영역으로 포커스가 이동하지 않도록 접근성을 강화했습니다.

  • insight 4

    모바일 환경 접근성 최적화와
    TalkBack, VoiceOver 대응

    모바일 중심 접근성 프로젝트로, 구글 크롬 기반 모바일 브라우저에서 작업을 진행하고, TalkBack(Android)과 VoiceOver(iOS) 환경에서 사용자 경험을 검증했습니다.

    모바일 브라우저 환경을 기준으로 접근성을 최적화하여, 화면 크기와 터치 조작 환경에서도 기능이 원활하게 동작하도록 구현했습니다.

    TalkBack과 VoiceOver를 통해 실제 모바일 사용자 환경에서 포커스 이동과 ARIA 속성 적용을 검증하고, 모바일 사용자의 접근성과 편의성을 확보했습니다.

Frontend
Project

  • step1

    Main Page

    처음 시작하는 메인 페이지에서는 여행 계획을 만들기 위한 출발점입니다. '계획 만들기' 버튼을 클릭하면 다양한 질문을 통해 나에게 딱 맞는 여행을 설계할 수 있는 질문 페이지로 이동합니다. 직관적인 디자인과 간단한 안내로 처음 방문한 사용자도 쉽게 시작할 수 있도록 구성했습니다.

  • step2

    Question Page

    여행 스타일, 음식 취향, 활동 선호도 등 다양한 질문에 답하면서 나만의 여행 취향을 파악하고, 그에 맞는 여행지를 추천받을 수 있도록 구성된 맞춤형 질문 페이지입니다. 사용자의 선택에 따라 결과가 달라지는 인터랙티브한 흐름을 통해 참여감을 높였습니다.

    반응형
  • step3

    Result Page

    질문에 대한 응답을 기반으로 추천된 여행지를 확인할 수 있으며, 각 장소의 상세 정보를 구글 지도에서 바로 볼 수 있습니다. 마음에 드는 장소는 장바구니에 담아 나중에 계획에 반영할 수 있습니다. 실제 위치와 함께 제공되는 시각적인 정보로 여행지를 보다 생생하게 확인할 수 있습니다.

  • step4

    Storage Page

    장바구니에 담은 장소들을 기반으로 만들어진 최종 여행 계획을 확인할 수 있는 페이지입니다. 일정별로 정리된 여행 코스를 한눈에 확인하고, 직접 세운 여행 계획을 공유할 수 있습니다. 나만의 여행 루트를 시각적으로 구성하여 쉽게 계획을 점검하고 수정할 수 있도록 했습니다.
